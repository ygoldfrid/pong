# Yaniv's Pong

A simple version of the all-time classic arcade game

## Try it online

You can play it [here](https://pong.yanivgoldfrid.com)

## Commands

Menu:
- Up: ArrowUp
- Down: ArrowDown
- Select: SpaceBar

1 Player:
- Up: ArrowUp or W
- Down: ArrowDown or S
- or just use the mouse

2 Players:
- Player 1
  - Up: W
  - Down: S
- Player 2
  - Up: ArrowUp
  - Down: ArrowDown

Pause the Game: P

## Built With

JavaScript + HTML5
